# TT Vpn

macOS için taşınabilir, menü-çubuğu VPN istemcisi. openconnect (GlobalProtect/Fortinet/AnyConnect),
OpenVPN ve strongSwan (IKEv1/IPsec) motorlarını uygulamanın içine gömer — **Homebrew gerekmez**.
Apple Silicon + Intel, macOS 13 (Ventura) ve üzeri.
